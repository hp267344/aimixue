1、学习node.js如何一条龙

	前端(get请求)->后端
	  不管是网址输入还是是表单发起的请求，还是ajax发起的请求，服务器端始终只需要一句话
		const url=require("url");
		var objUrl=url.parse(req.url,true);
		objUrl.query.键;//获取到值

	后端->前端
	  服务器端想要发到前端：res.end(JSON.stringif(JSONObj));

	  前端想要接住：ajax

	后端->数据库（增删改）&数据库->后端（查）：
	  不需要记忆：只需要看readme
	1、下载引入
	var mysql=require('mysql');

	2、创建链接数据库对象
	var conn= mysql.createConnection({
  		host     : '127.0.0.1',
  		user     : 'root',
  		password : '',
  		database : 'h52003'
	});
	

	3、创建SQL语句
	var sql="DELETE FROM user WHERE id=2";

	4、执行SQL语句
	conn.query(sql, function (error, results) {
  		console.log(results);
		//只要是增删改：OK
		//查：JSON对象
	});

	5、断开和数据库的链接
	conn.end();	

	一条龙：前端->服务器端->数据库（注册）
		数据库->服务器端->前端
